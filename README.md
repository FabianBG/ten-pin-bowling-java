# TEN PIN BOWLING CODE CHALLENGE JOBSITY 
Project wich solves the ten pin bowling code challenge of Jobsity. Developed in Java and Gradle.

## Project structure

```
├── build.gradle
├── gradle    
│   └── wrapper
│       ├── gradle-wrapper.jar
│       └── gradle-wrapper.properties
├── gradlew
├── gradlew.bat
├── settings.gradle
└── src
    ├── main
    │   ├── java  
    │   │   └── demo
    │   │       └── App.java
    │   └── resources
    └── test      
        ├── java
        │   └── demo
        │       └── AppTest.java
        └── resources
```

## Requirements

* Java >= 8
* Gradle
* JUnit
